--------------------------------------------------------------------------------------------------------
Данный проект — это веб-приложение «Онлайн каталог товаров», выполненное в формате интернет-магазина.

Приложение позволяет:

искать товары по названию;

просматривать список товаров;

открывать страницу конкретного товара с подробной информацией;

добавлять товары в избранное.

Поиск сопровождается подсказками при вводе текста.
Все данные подгружаются автоматически из тестового сервиса и не требуют настройки базы данных.

Приложение работает в браузере, адаптировано под мобильные устройства и не требует сложной установки — достаточно запустить проект на компьютере и открыть его по локальному адресу.
-------------------------------------------------------------------------------------
Как запустить проект

Установите Node.js (LTS) с сайта https://nodejs.org
 и перезагрузите компьютер.

Распакуйте архив с проектом в любую папку.

Откройте эту папку и убедитесь, что внутри есть файл package.json.

Откройте терминал в папке проекта (Shift + ПКМ → Открыть терминал).

Выполните команду:

npm install


После установки выполните:

npm run dev


Откройте в браузере адрес, который появится в терминале
(обычно http://localhost:5173).

Готово. Проект работает в браузере.
